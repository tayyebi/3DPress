3DPress lets you accept 3D printing orders with file upload, STL preview, material selection, and WooCommerce integration. Includes Elementor widget and multi-step form.

== Description ==
3DPress is a WordPress plugin for 3D printing businesses. It allows users to upload 3D model files (STL/OBJ), preview them, select materials, enter model details, and get instant cost/time estimates. Admins can configure materials, pricing, and optionally connect to WooCommerce for checkout, shipping, and tracking. Includes Elementor widget and shortcode support.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/3dpress` directory, or install through the WordPress plugins screen.
2. Activate the plugin through the 'Plugins' screen.
3. Configure materials, pricing, and WooCommerce integration in the 3DPress settings.
4. Use the `[three_dpress_form]` shortcode or the Elementor widget to display the order form.

== Frequently Asked Questions ==
= Does it support WooCommerce? =
Yes, you can link orders to WooCommerce for payment, shipping, and tracking.

= Can I use it with Elementor? =
Yes, 3DPress provides an Elementor widget for easy drag-and-drop form placement.
